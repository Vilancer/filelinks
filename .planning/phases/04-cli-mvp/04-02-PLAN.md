1. Resolve **`cwd`** from global options (default **`process.cwd()`**).
2. Load config via **`loadFileLinksConfig(cwd, { configPath: opts.config })`** when **`--config`** is set; else **`loadFileLinksConfig(cwd)`**.

5. **Human mode:** print lines to **stdout** (**D-06**, **D-08**) including **trigger**, **affected path**, **reason**, **effective severity**.
6. **`--json` mode:** print a **JSON array** (or `{ "violations": [...] }` — pick one and use consistently) to **stdout** with stable field names documented later in README.
7. **Exit code:** **1** if **any** printed/effective row has **`error`**; **0** otherwise (**D-03–D-05**). Warnings must still appear (**D-05**).

Refactor **`formatters.ts`** for shared JSON / text helpers if needed.
